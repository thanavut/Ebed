# Ebed System

Lean, audit-first web application for real-world use and long-term sustainability.

## Folder structure
- `backend/schema/` → JSON schemas for all entities
- `frontend/mockups/` → UI layout and design previews
- `export/` → Sample output formats for reports and data

## Design Principles
- Transparent and traceable
- Consistent UI (beige / gray / blue)
- Never lock out admin control
- Always portable and auditable
